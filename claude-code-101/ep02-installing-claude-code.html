Installing Claude Code

Claude Code 101 系列教程 · 第 2 课 · 来自 Claude 官方频道

终端安装

Claude Code 的安装非常简单，无论你想在终端、Web 还是 IDE 中使用。

macOS / Linux / WSL

使用一条 curl 命令即可安装：

# 推荐：一条命令安装（自带自动更新）
curl ...

# 或者使用 Homebrew
brew install claude

注意： Homebrew 安装方式不支持自动更新。

Windows

# PowerShell
Invoke-RestMethod ...

# CMD 也可以使用 curl
curl ...

# 或使用 winget
winget install claude

同样，winget 也不支持自动更新。

首次启动

进入你的项目目录，运行 claude，按照指引完成初始化：

  1. 选择配色主题
  2. 登录 Claude 账户（Pro、Max 或 Enterprise）
  3. 或者使用 API Key 登录
  4. 如果你的组织有 Claude Enterprise 账户，请选择该选项

重要： Claude Code 对启动目录及其所有子文件夹拥有访问权限。

Visual Studio Code 安装

  1. 打开 VS Code 的 扩展面板
  2. 搜索 Claude Code
  3. 找到 Anthropic 发布的扩展（确认有蓝色认证标记）
  4. 点击 安装
  5. 安装后可能需要重启 VS Code

启动方式

  • 使用快捷键 Ctrl/Cmd + Shift + P 打开命令面板，搜索 Claude Code → 打开新标签页
  • 如果文件已打开，也可以直接点击 Claude 图标
  • 你也可以在设置文件中关闭 UI，直接使用终端体验

JetBrains IDEs 安装

  1. 从 JetBrains Marketplace 安装 Claude Code 插件
  2. 安装后重启 IDE
  3. 重新打开后，你会看到 Claude 图标
  4. 点击后会打开一个面板，提供终端体验，可以与你的工作流并行使用

Claude Desktop 安装

  1. 安装 Claude Desktop 并登录
  2. 在顶部你会看到一个 Code 开关
  3. 点击即可启动 Claude Code

外观和体验类似 Claude Chat，但允许你：

  • 在特定文件夹中工作
  • 修改权限设置
  • 甚至在云端环境中工作

Web 版

访问 claude.ai/code 即可使用。

功能与桌面应用类似，但仅限于 GitHub 仓库。

各平台对比

平台 特点 适合场景
终端 功能最全、更新最快 日常开发、第一时间体验新功能
IDE 集成 与代码编辑器深度融合 喜欢在编辑器内完成一切的开发者
Desktop 可后台运行 让 Claude 在后台工作的同时处理其他任务
Web 远程访问 GitHub 仓库 远程协作、多会话并行

选择建议： 无论你选择哪种方式，Claude Code 的核心体验是一致的。选择最适合你工作流的方式即可。
